Terms:

[message] - message to be sent to other processes

[inbox] - message received from other processes

[priority lists] - three linked lists of ready state PCBs separated by their priority

[findhelper] - utility for finding a PCB in priority lists

[killhelper] - utility for finding a PCB in priority lists and killing it

[runningProcess] - pointer to the current running process

[loadProcess] - finding a successor process in the priority lists and loading it into the runningProcess

[PID] - Process ID

[SID] - Semaphore ID
